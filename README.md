# miniORM
a non bloat ORM (Object Relational Mapper) for small/fast PHP Projects
PHP 5.6 is required.
miniORM supports MySQL, SQLite, PostgreSQL and even basic Oracle. 
miniORM base of the convention-over-configuration Philosophy
